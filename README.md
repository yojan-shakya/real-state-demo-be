# About the project

This project is a part of techKrafts assignment.

# How to run the app

1. Please make sure port 3000 is not in use
2. Create .env.docker (can copy from .env.docker.example)
3. `docker compose up`

# API example

1. Run the app.
2. Go to `localhost:3000/api` for swagger

# Test

1. `npm run test`

# Notes

1. Delay interceptor is used to simulate network delays
2. put x-admin=true in header to simulate admin priveleges (in listings/id)
